Java Password Manager

A secure password manager built in Java.

Features
- AES encryption (CBC mode with IV)
- Master password authentication
- Add / View / Edit / Delete credentials
- File-based secure storage

Security Concepts Implemented
- Symmetric encryption (AES)
- Key derivation using SHA-256
- Initialization Vectors (IV) for randomness

Tech Stack
- Java
- File I/O
- Cryptography APIs
